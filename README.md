# Offlens model packs

On-demand model packs downloaded by the [Offlens](https://github.com/owlzworks/offlens) app at first use. Packs are distributed as GitHub Release assets.

| Pack | Model | License | Size |
|---|---|---|---|
| face-restore v1 | GFPGAN v1.4 (fp16) | Apache-2.0 — © Tencent ARC ([GFPGAN](https://github.com/TencentARC/GFPGAN)) | 170 MB |

The fp16 conversion is byte-checked against fp32 (PSNR 74 dB on reference inputs). Each release lists the SHA-256 the app verifies after download.
